# NASA Space Mission Classification AI
Final Project – Aima – ITAI2372  
Track: Implementation

## Description
This project builds a simple AI model that classifies NASA missions as "Manned" or "Unmanned" based on mission name, launch year, destination, and mission goal. The dataset is generated inside the code, so no external files are needed.

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the project:
   python src/main.py

## Files
- src/main.py – main program
- src/model.py – machine learning model
- src/preprocess.py – preprocessing steps
- src/dataset.py – synthetic dataset
- docs/Documentation.md – full project documentation
- slides/FinalPresentation.txt – presentation script
